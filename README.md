# 計算表作成サイト
電卓でちょっとした計算をしている時、さっき使った数値をまた使いたくなった事がありませんか？<br>
でもExcelとか使うほどじゃ…って事が多いなら、是非このサイトを利用してみて下さい！！

![](calculatorpage.png)

## 出来る事
・キャンバス上に数値を配置し、簡単な四則演算を使用した計算表が作成出来ます。<br>
・掲示板から他人が作成した計算表を使用出来、また自身の作成した計算表を掲示板へ投稿して共有する事も出来ます。
